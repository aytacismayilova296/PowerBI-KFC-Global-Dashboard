# PowerBI-KFC-Global-Dashboard
This project presents an interactive Global Sales Performance Dashboard for KFC, built using Power BI. The dashboard analyzes sales performance, customer activity, marketing effectiveness, and branch performance across multiple countries and years.
The dataset contains transactional sales records by month, branch, and country. It was used to explore trends, compare regional performance, and identify high-performing locations.

## Dataset Overview
The dataset includes the following key fields:
• Year — Reporting year
• Month — Reporting month
• Country — Country where the branch operates
• Branch_ID — Unique branch identifier
• Sales — Total sales amount
• Customers — Number of customers served
• Marketing_Spend — Marketing expenditure

## Tools Used: 
•	Power BI
•	DAX
•	Data transformation and cleaning in Power Query.

## Dashboard Features
The dashboard includes the following key visualizations:
• KPI Cards showing total sales, customers, marketing spend, and average sales
• Sales trend over time
• ROI by year
• Top-performing branches by sales
• Sales distribution by country
• Interactive slicers for filtering by year, country, and branch

## Key Insights
• Sales show noticeable fluctuations across years with a strong upward trend in recent periods
• Certain branches consistently outperform others in total sales
• Sales distribution varies across countries, indicating different market strengths
• Marketing investment correlates with higher sales in several periods

## Data Preparation
Before building the dashboard, the dataset was cleaned and transformed:
• Verified data types
• Structured columns for analysis
• Aggregated measures for KPIs
• Created calculated metrics for performance comparison

## Notes
This project was created for learning and portfolio purposes. The dataset used does not represent any official or confidential company data.
